Question-1:What are the different ways to select an element in the Dom?

We can select an element using its id with getElementById(), by class with getElementsByClassName(), or by tag name with getElementsByTagName().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML works with HTML content, including tags. innerText takes CSS styles into account. textContent is typically faster and preferred.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is attached to a parent element. This allows for more efficient event handling, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on a child element propagates upwards through its parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

we can create elements with document.createElement(), add them using appendChild(), and remove them with removeChild() or remove() in JavaScript.
